About The Project

3 days ago by Noy Miran

Anomaly detection project wrote in Scala and using Play Framework application with ReactiveMongo, Macwire and SBT build tool

Project Description

Anomaly detection generic system on graphs that gives alerts and information on anomalies from the data.

What's the need?

For a Big Data AdTech company like Fyber, real-time data analytics and insights generation are highly valuable (for both technical and business considerations).

Anomaly detection helps companies determine when something changes in their normal business patterns.
When done well, it can give a company the insight it needs to investigate the root cause of the change, make decisions and take actions that can save money (or prevent losing it).
With an anomaly detection system, we could identify software bugs, capture seasonal patterns, detect network latency etc.
